Дистрибутив должен быть размещен в корне диска "C:"


Содержимое каталогов дистрибутива

  1. components - составные компоненты дистрибутива, участвующие в установке

  2. qbs - проекты

  3. qbs-build - скомпилированные проекты

  4. tools - вспомогательные инструменты

  5. docs - документация на контроллер


Порядок установки

  1. Ознакомиться с описанием

  2. Принять условия лицензионного соглашения

  3. Установить QtCreator

  4. Установить драйвер

  5. Установить Python 2.7

  6. Редактировать переменную PATH

  7. Настроить инструментарий разработки под ARM на QtCreator.

  8. Внимательно изучить руководство.

Содержимое каталогов дистрибутива

  1. components - составные компоненты дистрибутива, участвующие в установке

  2. qbs - проекты

  3. tools - вспомогательные инструменты

  4. docs - документация на контроллер

Порядок установки

  1. Ознакомиться с описанием

  2. Принять условия лицензионного соглашения

  3. Установить QtCreator

  4. Установить драйвер JLink

  5. Установить Python 2.7

  6. Отредактировать переменную PATH

  7. Настроить инструментарий разработки ARM.

  8. Изучить руководство по QtCreator.

Нельзя перемещать или удалять компоненты

  1. cmake

  2. gcc-arm-none-eabi

  3. openocd

  4. MDR32F9Q1.svd

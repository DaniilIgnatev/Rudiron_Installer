Каталоги

  1. components - пакеты дистрибутива, участвующие в установке.

  2. installer - установщик дистрибутива.

  3. tools - вспомогательные инструменты.

  4. docs - сопроводительная документация.

  5. Arduino_Core_Rudiron - ардуино-совместимая библиотека.

  6. Rudiron_Projects - шаблоны и примеры проектов.

Инструкция по установке

  1. Ознакомьтесь со слайдами, пролистайте их до конца.

  2. Нажмите кнопку "Установить".

  3. Следуя примерам на слайдах, повторите указанные действия.

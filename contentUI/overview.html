На каждом этапе установки дистрибутива

  1. Ознакомьтесь с инструкцией, пролистайте ее до конца.

  2. Нажмите кнопку "Установить".

  3. Следуя указаниям и примерам на слайдах, повторите все действия.

Содержимое каталогов дистрибутива

  1. components - составные компоненты дистрибутива, участвующие в установке.

  2. qbs - шаблоны проектов для платы.

  3. tools - вспомогательные инструменты.

  4. docs - документация на плату.

Порядок установки

  1. Принять условия лицензионного соглашения.

  2. Установить Qt Creator.

  3. Установить драйвер JLink, при наличии программатора.

  4. Заменить драйвер на WinUSB через Zadig, при наличии программатора.

  5. Установить Python 2.7.

  6. Настроить инструментарий разработки.

  7. Изучить руководство пользователя.

Нельзя перемещать или удалять компоненты

  1. cmake.

  2. gcc-arm-none-eabi.

  3. openocd.

  4. MDR32F9Q1.svd.


Программирование возможно как через программатор, так и через виртуальный COM порт.

Отладка возможна только при наличии JLink-совместимого программатора.

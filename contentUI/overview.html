Отладка возможна только при наличии JLink-совместимого программатора.

Программирование возможно как через программатор, так и через виртуальный COM порт.

Содержимое каталогов дистрибутива

  1. components - составные компоненты дистрибутива, участвующие в установке

  2. qbs - проекты

  3. tools - вспомогательные инструменты

  4. docs - документация на контроллер

Порядок установки

  1. Ознакомиться с описанием

  2. Принять условия лицензионного соглашения

  3. Установить Qt Creator

  4. Установить драйвер JLink, при наличии программатора

  5. Заменить драйвер на WinUSB через Zadig, при наличии программатора

  6. Установить Python 2.7

  7. Настроить инструментарий разработки

  8. Изучить руководство пользователя

Нельзя перемещать или удалять компоненты

  1. cmake

  2. gcc-arm-none-eabi

  3. openocd

  4. MDR32F9Q1.svd
